# 📝 To-Do List App

A simple and beginner-friendly command-line To-Do List application built with Python.

## 🚀 Features

✅ View Tasks  
➕ Add New Tasks  
🗑️ Delete Tasks  
💾 Save Tasks to a File  
📂 Load Tasks Automatically on Startup

## 🛠️ Technologies Used

- 🐍 Python 3
- 📄 File Handling
- 🔄 Loops
- ⚡ Functions
- 🚨 Exception Handling

## 📁 Project Structure

## ▶️ How to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/todo-list-app.git
```

### 2️⃣ Open the Project Folder

```bash
cd todo-list-app
```

### 3️⃣ Run the Application

```bash
python main.py
```
